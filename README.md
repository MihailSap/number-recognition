## Система распознавания чисел и цифр

**Выполнил:** Сапегин Михаил

**Для работы с проектом нужно:**
1. Установить необходимые библиотеки
2. Запустить скрипт model.py для создания и обучения модели. В результате, должна сохраниться обученная модель (digit_cnn.h5)
3. Запустить скрипт recognition_digit.py или recognition_numbers.py с указанием папки, в которой лежат изображения, которые необходимо распознать
4. Запустить скрипт experiment для получения результатов зависимости вероятности правильного распознавания от отношений сигнал/шум и уверенность/шум
